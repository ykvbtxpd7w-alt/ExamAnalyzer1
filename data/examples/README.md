# Приклади та legacy-дані

Файли в цій папці не використовуються поточним UI.

- `question_bank.json` — старий формат білетів з `base_complexity` + `q_type`
- `questions.json` — мінімальний приклад плоского списку питань
- `tickets.json` — порожня заготовка під збережені білети (замінена історією через `seed`)

Активні банки предметів: `data/math_analysis.json`, `data/physics.json`.

Ми працюємо над курсовим проєктом ExamAnalyzer. Це програмна система, яка аналізує складність екзаменаційних білетів. Основна ідея проєкту — формалізувати оцінку складності білетів за допомогою математичної моделі.
Білет складається з кількох питань. Для кожного питання задаються:

базова складність (число від 0 до 1),
тип питання (теорія, задача, доведення).
Кожен тип має ваговий коефіцієнт:

theory — 1.0
problem — 1.3
proof — 1.6
Складність питання обчислюється як:
S = base_complexity × type_weight
Складність білета обчислюється як середнє значення складностей усіх питань:
Score_ticket = (1/n) × Σ(base_i × weight_i)
Система повинна:

зчитувати білети з CSV або JSON,
обчислювати складність білетів,
порівнювати білети між собою,
будувати статистику (середнє, дисперсія),
показувати рейтинг білетів за складністю.
Проєкт робиться командою приблизно з 5 людей у межах семестру (~3 місяці).
